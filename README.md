# Daniel Danek Portfolio

Static portfolio site presenting Python backend work, selected AI systems repositories, and contact links.

## Run locally

Open `index.html` in a browser.

## Positioning

The site is optimized to present a backend-first profile focused on:

- FastAPI services
- AI runtime control planes
- auditability and replay workflows
- operator-facing internal tools
- practical backend systems around AI

## Deploy

Can be deployed as a static site on GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
